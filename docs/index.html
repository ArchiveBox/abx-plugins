ArchiveBox ecosystem, plugins, and web archiving context

The web rots fast. News stories get edited, videos disappear, social posts vanish, and cited links break within a few years. ArchiveBox is an open-source self-hosted web archiving platform built to help individuals and organizations keep durable local copies of the web in standard formats like HTML, JSON, TXT, PDF, PNG, SQLite, and WARC.

This marketplace documents the shared abx-plugins catalog used by ArchiveBox and abx-dl. ArchiveBox manages full collections and long-term archives, abx-dl runs plugin pipelines directly, abx-pkg installs supporting binaries, and abx-plugins defines the hook scripts and config schemas that actually crawl pages, install tools, and extract per-site outputs.
